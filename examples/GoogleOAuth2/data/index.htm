ESP Google API library


This is a demo page for Arduino-Google-API library.
First of all, take a look at the readme on repository page to setup properly your system.

If you are running ESP_GoogleDrive example, load this page /drive.htm to test some of the functionality of library.

For the example ESP_GoogleEmail check this page /email.htm and send an email from your ESP device with data passed from browser as GET request. If you need to authorize the application, go to page /token.htm and follow istructions.
